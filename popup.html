Steemit More Info Features

Votes value

Adds the amount in dollars of the votes a post has received, so that you can see how much each account's vote affected the rewards

Vote Tab

Adds a tab where you can see the incoming and outgoing votes of an account

External Links Tab

Adds a tab where you can find external links related to Steemit

Search Bar

Change the search bar to a fast overlay

Mentions Tab

Adds a tab where you can see all the posts in which an user has been mentionned

Vote Slider Value

Adds the amount in dollars your vote is worth to the voting/downvoting slider (Only for account with enough SP)

Vote Slider Value (for busy)

Adds the amount in dollars your vote is worth to the voting/downvoting slider (Only for account with enough SP). This feature will only be used on Busy. Unlike classic vote slider value, this feature for Busy need connection to SteemConnect

Followers Table

Replace the followers and following page with a table based list of followers/following accounts, showing more information, making the list sortable and with a quick way to follow/unfollow

Boost Button

Add a Boost button next to the Promote button to the posts to easily send SBD to @minnowbooster for boosting upvotes

Gif Picker

Tool to easily add GIF from giphy.com to posts and comments

User Information Popover

Add a popover to the banner to display information about your account (Voting Power, max vote value, full in ... hours)

Blog Histogram

Show an histogram of the account's posts history on the blog page of an user

MarkDown Editor Beautifier

When you create a post, the preview and the editor are side by side and scroll together to show you the markup code and the preview of the same area (works only for screen wider than 900px)

Float Post's Voting Bar

While reading a post, this feature keeps the voting bar alwasy visible on the top of the page so that it is easier to upvote and look at more information about the post
